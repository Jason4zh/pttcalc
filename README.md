# pttcalc

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一款轻量、无后端的 Arcaea PTT（Potential）计算器，支持全难度歌曲成绩录入、实时PTT计算、B30/B10 综合评分统计，数据本地存储不丢失。

## 🚀 在线访问
已部署至 GitHub Pages，可直接在线使用：
👉 [Arcaea PTT 计算器](https://jason4zh.github.io/pttcalc/)

## ✨ 核心功能
- 🎵 **全难度覆盖**：支持 Past/Present/Future/Beyond/Eternal 所有难度筛选
- ⚡ **实时计算**：输入分数自动计算对应 PTT 值与评级（EX+/EX/AA/A/B/C/D）
- 💾 **本地存储**：成绩数据保存在浏览器 localStorage，刷新/关闭页面不丢失
- 📤📥 **数据迁移**：支持 `.pttdat` 格式成绩导入/导出，方便备份与跨设备同步
- 🔍 **精准检索**：按歌曲名/艺术家关键词快速筛选歌曲
- 📊 **多维度排序**：支持按标题、定数、分数、PTT 值升降序排序
- 📈 **自动统计**：实时计算 B30（前30佳PTT）+ B10（前10佳PTT）综合评分

## 📦 快速开始
### 本地运行
1. 克隆仓库
   ```bash
   git clone https://github.com/Jason4zh/pttcalc.git
   cd pttcalc
   ```
2. 直接打开 `index.html` 文件（现代浏览器即可，无需额外环境）

### GitHub Pages 部署
1. 将项目文件推送到 GitHub 仓库
2. 进入仓库 → `Settings` → `Pages`
3. `Build and deployment` 模块选择：
   - Source：`Deploy from a branch`
   - Branch：`main`（或你的分支名）→ `/ (root)`
4. 等待部署完成，访问仓库生成的 Pages 地址即可

## 📋 使用指南
### 基础操作
1. **难度筛选**：点击页面上方「筛选难度」区域的单选按钮，切换展示对应难度歌曲
2. **搜索歌曲**：在搜索框输入歌曲名/艺术家关键词，列表实时过滤匹配结果
3. **录入分数**：在歌曲行的分数输入框填写成绩（0-10000000），自动计算并展示 PTT 值和评级
4. **排序切换**：点击「排序」按钮，循环切换排序方式（默认/标题/定数/分数/PTT，支持升降序）

### 数据管理
- **导出成绩**：点击「导出成绩」按钮，下载 `.pttdat` 格式的成绩备份文件（命名包含时间戳）
- **导入成绩**：点击「导入成绩」按钮，选择本地 `.pttdat` 文件，自动合并/覆盖成绩数据

### PTT 计算规则
严格遵循 Arcaea 官方逻辑：
| 分数区间          | 评级  | PTT 计算公式                          |
|-------------------|-------|---------------------------------------|
| ≥10000000         | EX+   | 定数 + 2                              |
| 9900000 - 9999999 | EX+   | 定数 + 1 + (分数-9800000)/200000      |
| 9800000 - 9899999 | EX    | 定数 + 1 + (分数-9800000)/200000      |
| 9500000 - 9799999 | AA    | 定数 + (分数-9500000)/300000          |
| 9200000 - 9499999 | A     | 定数 + (分数-9500000)/300000          |
| 8900000 - 9199999 | B     | 定数 + (分数-9500000)/300000          |
| 8600000 - 8899999 | C     | 定数 + (分数-9500000)/300000          |
| >0 - 8599999      | D     | 定数 + (分数-9500000)/300000          |

**总 PTT 计算**：`(B30 总分 + B10 总分) / 40`

## 📁 文件说明

## ⚠️ 注意事项
1. 数据存储在浏览器 localStorage，清除浏览器缓存/数据会导致成绩丢失，建议定期导出备份
2. 仅支持 `.pttdat` 格式导入文件，其他格式会提示错误
3. 分数输入范围为 0-10000000，超出范围会自动按边界值处理
4. 歌曲数据基于 Arcaea 6.11.8f 版本，如需更新请替换 `songlist6_11_8f.json` 和 `grade6_11_8f.json`
